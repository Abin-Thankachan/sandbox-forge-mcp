Frequently Asked Questions

What is MCP? expand_more
The Model Context Protocol (MCP) is an open standard that enables AI models to interact with local tools and data securely. SandboxForge is a specialized server for this protocol.
Do I need Docker installed on host? expand_more
No. SandboxForge uses Lima to provision an isolated Linux VM that contains its own Docker engine, keeping your host system clean.
Is Linux supported? expand_more
Yes, Linux is fully supported. On Linux, SandboxForge can run natively or via Lima depending on your configuration preferences.
What happens if dependencies are missing? expand_more
SandboxForge performs a pre-flight check. If dependencies like Lima are missing, it provides a structured error with the exact shell command needed to install them.
